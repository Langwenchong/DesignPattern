代码仓库一共有6个分支，每一个分支对应着不同的设计模式分类，详情如下：

|  分支名称  |           存储文件类型           |
| :--------: | :------------------------------: |
|  Creation  | 创建型设计模式课程参考代码（5）  |
| Structural | 结构型设计模式课程参考代码（7）  |
